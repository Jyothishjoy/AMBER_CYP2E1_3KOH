1. Follow the steps in https://pmc.ncbi.nlm.nih.gov/articles/PMC4669947/
2. Download and install the packages mentioned in the paper.
3. Don't forget to copy autogrid4.exe and autodock4.exe into the current working directory.
4. This will make their execution much easier through WSL2 (Ubuntu).
5. Use WSL2 to navidate to the files and execuitables located.
6. Run [./autogrid4.exe -p a.gpf -l a.glg &] and [./autodock4.exe -p a.dpf -l a.dlg &] as described in the paper.
7. Everyting should work fine.
